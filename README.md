Shadowsocks软件介绍
SS，全称为 Shadowsocks（下文简称为SS），是一种基于 Socks5 代理方式的网络数据加密传输包的工具，可以使用其于科学上网 也就是翻墙 ，是目前最稳定，最常用的科学上网工具之一，全平台支持

SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

2017年7月19日，ShadowsocksR作者breakwa11在Telegram频道ShadowsocksR news里转发了深圳市启用SS协议检测并被大量用户转发，引发恐慌。7月24日，breakwa11发布了闭源的SS被动检测程序，引发争议。 2017年7月27日，breakwa11遭到自称 “ESU.TV” 的不明身份人士人身攻击，对方宣称如果不停止开发并阻止用户讨论此事件将发布更多包含个人隐私的资料，随后breakwa11表示遭到对方人肉搜索并公开个人资料的是无关人士，为了防止对方继续伤害无关人士，breakwa11将删除GitHub上的所有代码、解散相关交流群组，停止ShadowsocksR项目。但项目已被多人fork。并有人在其基础上继续发布新的版本，例如SSRR

Shadowrocket 使用指南：扫描、节点、分组与插件
一、Shadowrocket 扫描功能
Shadowrocket 提供了扫描功能，用户可以通过它自动寻找可用的代理节点。扫描步骤如下：

打开应用：启动 Shadowrocket 应用。
选择“扫描”选项：在主界面中找到并点击“扫描”按钮。
等待结果：应用会自动搜索可用的代理服务器，并列出扫描结果。
此功能可以帮助用户快速找到可用的节点，节省手动输入的时间。

二、Shadowrocket 节点
节点是指代理服务器的具体地址，用户在使用 Shadowrocket 时需要配置节点信息。添加节点的步骤：

进入配置界面：在应用内选择“配置”。
添加新节点：输入节点的服务器地址、端口号和认证信息。
保存设置：确认保存，以便后续使用。
通过多个节点的配置，用户可以轻松切换以获得更好的连接速度。

三、Shadowrocket 分组管理
为了便于管理，用户可以将不同的节点进行分组。分组功能允许用户按需选择不同的代理。设置分组的步骤：

进入分组设置：在应用中选择“分组”选项。
创建新分组：添加不同的分组名称。
分配节点：将所需的节点拖放到对应的分组中。
这种管理方式使得用户可以更有条理地使用多个节点。

四、Shadowrocket 插件
Shadowrocket 支持多种插件，以增强应用功能。例如，用户可以使用 DNS 解析插件以改善访问速度和安全性。安装插件的步骤如下：

下载插件：通过可信的渠道获取插件文件。
导入插件：在 Shadowrocket 应用中选择“插件”选项，导入下载的插件。
启用插件：根据需要启用相应的插件。
使用插件可以根据用户的需求进行个性化设置。

五、Windows 版本的 Shadowrocket
目前 Shadowrocket 主要针对 iOS 设备，官方并未推出 Windows 版本。对于 Windows 用户，可以考虑使用其他类似的代理工具，如 V2RayN 或 Shadowsocks 客户端，这些工具可以提供类似的功能。

六、Shadowrocket 与谷歌访问
用户在使用 Shadowrocket 时，可以方便地访问 Google 等被限制的网站。设置步骤：

确保节点可用：选择一个有效的代理节点。
配置 DNS：在设置中确保 DNS 配置正确，以便正常访问 Google 服务。
连接测试：通过浏览器访问 Google，验证连接是否成功。
